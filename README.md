# House Prices Prediction

## 📌 Problem Statement
House price prediction is an important real-world problem in real estate analytics.
The goal of this project is to predict house prices based on various property features
using machine learning regression techniques.

## 📂 Dataset
- Housing dataset (Kaggle / public dataset)
- Features include location, size, number of rooms, and other property attributes

## 🛠 Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 🔍 Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection and scaling
- Training regression models
- Model evaluation and comparison

## 🤖 Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## 📊 Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

## ✅ Results
The model predicts house prices with good accuracy and helps identify
important features influencing property prices.
